# 鑫办 - 智能办公协作平台

基于纯 HTML/CSS/JavaScript 开发的办公协作系统，可直接部署到 Cloudflare Pages。

## 功能特性

- 🏠 **工作台首页**：功能快捷入口，统计数据展示，最近公告
- 📋 **考勤打卡**：上班打卡，下班打卡，拍照打卡，请假申请
- ✏️ **审批中心**：发起审批，审批通过/驳回，按标签筛选
- 👥 **通讯录**：按部门分组，员工信息查看，快速发起聊天
- 💬 **消息聊天**：群聊/私聊，文字消息，图片消息
- 📢 **企业公告**：发布公告，查看历史公告

## 权限系统

- **超级管理员**：可查看全部数据，管理所有部门和员工，审批所有单据
- **部门管理员**：可管理本部门人员，审批本部门审批单据
- **普通员工**：仅查看本人数据，提交本人打卡，发起自己的审批

## 技术特点

- 纯前端实现，无需后端服务器
- 使用 localStorage 存储数据，刷新页面数据不丢失
- 响应式设计，支持移动端和桌面端
- 简洁美观的界面，仿钉钉风格
